# Cyberbot

## Project Overview
Cyberbot is a C# WPF desktop chatbot application developed for the Programming POE assessment.  
The application focuses on educating users about cybersecurity awareness through an interactive chatbot experience.

The chatbot allows users to ask questions related to cybersecurity topics such as:
- Password safety
- Phishing attacks
- Malware
- Online scams
- Privacy protection
- Safe browsing

The application also provides a modern graphical user interface with chatbot-style messaging and multimedia support.

---

## Features

### User Interaction
- Interactive chatbot conversation
- Personalised responses using the user’s name
- Friendly chatbot interface
- Message-based conversation layout

### Cybersecurity Topics
The chatbot can provide information and tips about:
- Password security
- Phishing
- Malware
- Online scams
- Privacy
- Safe browsing

### GUI Features
- WPF graphical interface
- Styled chat messages
- User input textbox
- Send button functionality
- Images and media assets
- Greeting sound effects

### Technical Features
- Built using C# and WPF
- Uses `ObservableCollection` for live chat updates
- Dictionary-based response system
- Randomised responses for better interaction
- Event-driven programming

## How to Run the Project

### Requirements
- Visual Studio 2022 or later
- .NET Desktop Development workload installed

### Steps
1. Extract the ZIP folder.
2. Open the solution/project file in Visual Studio.
3. Build the solution.
4. Run the application using:
   - `F5` (Start Debugging), or
   - `Ctrl + F5`
